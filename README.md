# geobaza-zolnierze-praca-inzynierska - skrypty pomocnicze

Repozytorium zawiera skrypty wykorzystane w pracy inżynierskiej
pt. „Koncepcja i projekt bazy danych czasowo-przestrzennych dla postaci i wydarzeń historycznych”.

# Zawartość
- skrypty importu danych do bazy SQLite
- skrypty uzupełniania współrzędnych (GeoNames API)

# Informacje dodatkowe
Skrypty zostały wykorzystane do:
- importu danych źródłowych,
- uzupełniania brakujących współrzędnych,

Kody stanowią załączniki do pracy inżynierskiej.
